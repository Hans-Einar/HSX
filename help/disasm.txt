disasm <pid> [spec]

Disassemble instructions for the specified PID. `spec` can be an address (`0x1234`)
or a symbol name. Options:

  --count N   Number of instructions (default 16)
  --source    Include source lines when provided by the executive

Output highlights the current PC with `=>` and annotates each instruction with
symbol names (`<function+offset>`) plus any available source locations.
