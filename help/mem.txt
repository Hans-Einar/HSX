mem <subcommand>

Subcommands:
  regions <pid>          List memory regions (start, end, kind).
  read <pid> <addr> [--count N] [--format x|d|i|s]
  dump <pid> <start> <end>
    Hex dump with ASCII preview (16 bytes per row).

Shortcuts:
  x/<count><format><size> <pid> <addr>
    Examine memory using GDB-style syntax (e.g. `x/16xb 1 0x200`). Format tokens:
      count   = repeat count (default 1)
      format  = x (hex), d (decimal), i (instructions), s (string)
      size    = b/h/w/g (1/2/4/8 bytes)

Formats:
  x = hex words (default)
  d = signed decimal
  i = disassemble instructions
  s = ASCII string
